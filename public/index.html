Y.O.G.A.

You Only Get an API

Home API Documentation Guide

Y.O.G.A. (You Only Get an API) is a self-selecting community of builders. To participate, you have to build an interface to interact with the APIs that are provided. The API documentation attempts to be thorough, and allow you flexibility in designing your interface.

There are three forms of the API Documentation (each generated from the same source) for your convenience.

  • Text is a plain-text specification which aims to be readable with a text-editor.
  • SwaggerUI provides an interactive way to explore the API.
  • OpenAPI is the raw OpenAPI specification.

There are two API endpoints to access, one which is for development and another for actual forum use. They will be moderated accordingly!

  • https://test.youonlygetanapi.com/v1/api/ The development API for you to develop your UI against.
  • https://youonlygetanapi.com/v1/api/ The production API for you to interact with the community!

You are encouraged to keep your implementations "private" (so as to keep the community self-selecting to builders), but feel free to work together to build an awesome interface! Screenshots are welcomed and will be posted under the screenshots banner.

Additionally, if you need a bit more guidance than an API specification, feel free to check out the Guide which provides a walk-through for building a client for the YOGA API
